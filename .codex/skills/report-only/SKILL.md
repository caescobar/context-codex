---

REGLAS DURAS
- Prohibido crear/editar/borrar/mover archivos.
- Prohibido “ya lo arreglé”. Solo análisis.
- Si el usuario pide informe: siempre incluir al inicio:
  “Acciones ejecutadas: Ninguna (modo report-only)”.

FORMATO OBLIGATORIO
1) Alcance analizado (paths/archivos).
2) Hallazgos (high/med/low) con evidencia.
3) Riesgos (qué podría romperse).
4) Recomendaciones priorizadas.
5) Plan sugerido (sin ejecutarlo).

SI FALTA CONTEXTO
- Preguntar 1 sola vez con 2–3 opciones.
- No asumir.
